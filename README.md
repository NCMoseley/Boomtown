# Boomtown Starter

## Installation

Download the repo, then run:

```bash
cd client && npm install
```

## Start-up

```bash
npm run dev
```

## For Dev

From inside the `server` directory, run:

```
json-server --watch db.json -p 4000:
 for the json server
```

From inside the `client` directory, run:

```
npm start:
for the React Compiler
```
